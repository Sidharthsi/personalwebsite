My Projects

Here are some of the projects I've worked on:

Car Wash Management System

A comprehensive application designed to streamline and automate the operations of a car wash business. Key features include:

  • Customer Registration: Allows customers to create and manage their profiles.
  • Booking Management: Enables customers to schedule car wash appointments and view their booking history.
  • Service Tracking: Monitors the status of ongoing car wash services and provides real-time updates.
  • Reporting: Provides detailed reports on service usage, customer activity, and financial transactions.

Technologies used: PHP for backend, HTML, CSS, JavaScript for frontend, and SQL for database management.

Car Rental Management System

A web-based application developed using Python and Django, designed to manage the operations of a car rental business. Key features include:

  • Vehicle Management: Manage fleet availability and maintenance records.
  • Customer Booking: Customers can search for vehicles, make reservations, and view booking history.
  • Admin Dashboard: Overview of business operations including bookings, customer activity, and financial performance.

Technologies used: Python & Django for backend, HTML, CSS & Bootstrap for frontend, and SQL for database management.
